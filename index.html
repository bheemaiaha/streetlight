STREETLIGHT

Bicycle + Smartphone + Cardboard Projector

View on GitHub Download .zip Download .tar.gz

Introduction

Streetlight is smartphone app which turns your night bicycle ride into an exciting experience.

Assembly Instructions

  • Buy this Mini Cardboard Smartphone Projector (price with shipment cca $16 USD) and mount it on bicycle handlebars
  • Install Streetlight app into your smartphone
  • Adjust your smartphone display brigthness to maximum level
  • Make sure GPS and internet connectivity is enabled
  • Start Streetlight app and insert your smartphone into the Cardboard Projector
  • Enjoy your bicycle ride

Help, Questions, Suggestions

If you have questions or suggestions please send email to streetlight@google.com

Modify and Contribute

Streetlight source code is hosted on github. If you discover some bug, you are welcome to fix it and contribute via pull request (or at least create a github issue :). If you have some feature to add please create a github issue so that we can discuss it prior creating a pull request.
